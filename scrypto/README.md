# ✏️ Scrypto
## ¿Qué es Scrypto?
Scrypto es un lenguaje de programación que Radix está creando para permitir el desarrollo rápido y seguro de aplicaciones financieras descentralizadas (dApps) . Las dApps en Radix se forman a partir de 'componentes', también conocidos como contratos inteligentes, escritos en Scrypto que se ejecutarán en el entorno de ejecución Radix Engine v2 . Scrypto toma prestada en gran medida la sintaxis de Rust, pero incluya características específicas orientadas a activos para el desarrollo de aplicaciones DeFi basadas en un paradigma FSM (máquina de estado finito).
